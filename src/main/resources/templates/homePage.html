Book Manage System

This is a management system for personal books, mainly in order to facilitate the statistics of books and their favorite books to evaluate the lightweight website project. I hope it can provide some help for me or other people who love reading.

Under continuous update.

Project introduction

The BMS Project

The project is under construction with the following objectives

  1. Add a book to your reading list.
  2. Delete a book from your reading list.
  3. Look up all the books in the list.
  4. Each book on the list should include the title, author, and your description of the book.

The techniques used in this project are : springboot + thymeleaf + bootstrap + MyBatis-Plus
